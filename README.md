
  # ERP para loja de carros

  Este ERP tem o propósito de gerenciar veículos que entram e saem de uma loja, registrando seus documentos, e informações.

  # Como Configurar

  ### Configurando o Aplicativo (Linux)
  ```ps1
  python manage.py makeimigrations cars && python manage.py migrate
  ```
  ### Executando o Aplicativo
  ```ps1
  python manage.py runserver
  ```
